# Workspace Configuration
## Clawdbot Personal Assistant Setup

This folder contains the configuration files for the personal AI assistant workspace.

## Configuration Files

### Core Identity
- **`SOUL.md`** - Assistant personality and behavior guidelines
- **`IDENTITY.md`** - Basic identity information (name, avatar, etc.)
- **`USER.md`** - Information about the human user

### Behavior & Memory
- **`AGENTS.md`** - Core instructions for assistant behavior
- **`TOOLS.md`** - Local tool notes and preferences
- **`MEMORY.md`** - Long-term memory storage (private contexts only)
- **`HEARTBEAT.md`** - Periodic task checklist

### Data Storage
- **`memory/`** - Daily memory files and context storage
- **`notes/`** - Temporary notes and scratch work
- **`scripts/`** - Utility scripts
- **`downloads/`** - Downloaded files
- **`projects/`** - Small project folders
- **`canvas/`** - Canvas outputs and visualizations

## Usage

These files are automatically loaded by the Clawdbot assistant to:
- Understand personality and behavior expectations
- Remember context across sessions  
- Maintain user preferences and settings
- Store important information and decisions

## Privacy

- **`MEMORY.md`** contains personal context - only loaded in direct chats
- **Daily memory files** store session logs and important events
- **User information** helps the assistant provide better support
